# Just Javascript with webpack

1. Clone it
   ```sh
   git clone https://pokhiii@github.com/pokhiii/just-js-webpack.git
   ```

1. Take position
   ```sh
   cd just-js-webpack
   ```

1. Call your friends
   ```sh
   yarn
   ```

1. Set things up
   ```sh
   cp .env.example .env
   ```

1. Build it
   ```sh
   yarn build
   ```

1. Serve it
   ```sh
   yarn run serve
   ```
